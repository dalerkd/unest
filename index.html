obfuscate Machine Code to Anti-Analysis

多态混淆

多态指改变原代码形态,
混淆指改变原代码流程,
即在保留代码原功能前提下对代码进行变形.

目前支持

intel x86 32位指令集,
纯2制代码段格式,
COFF OBJ格式.

开放&&开源

是的，你没有看错，我们开放源代码,
你可以审查每一行代码以确保数据安全,
也可以为此项目贡献您的力量.

贡献力量

所有贡献将会获得感谢和回报.
